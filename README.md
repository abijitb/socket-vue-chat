# socket-vue-chat

Simple Chat Using Socket.io and Vue JS NodeJS ExpressJs.

### ScreenShot

![Chat Screen](src/assets/screen.png)

### Installing

run these command in terminal.

For FrontEnd, Go to Root Directory

```
npm install
npm run serve

```
For Server, Go to Server directory

```
npm install
node server.js

```
